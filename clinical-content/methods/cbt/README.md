# CBT content (write first)

Author CBT documents here, one `.md` file per document, named `<id>.md`, each copied
from the matching template in `../../templates/`.

See `../../METHODS_ROADMAP.md` §1 for the exact MVP set required before CBT is
considered releasable. In short:

- 1 × method overview (`cbt_overview`)
- 3–4 × psychoeducation pieces (cognitive model, thinking traps, avoidance loop,
  reassurance loop)
- 4–6 × intervention protocols (cognitive restructuring, evidence-for-and-against,
  behavioral activation, graded exposure, behavioral experiment, worry postponement)
- 4–6 × homework exercises (thought record, activity scheduling, exposure step,
  delay-the-reassurance, worry time)

Two fully worked CBT documents already exist in `../../examples/` — imitate them.
Use only taxonomy terms (`../../taxonomy.md`). Run the §9 checklist before submitting.
