# Psychodynamic / schema content (write last)

Author depth-oriented documents here (one `<id>.md` per document, from
`../../templates/`). See `../../METHODS_ROADMAP.md` §4. This is formulation-supporting
depth work and the least scripted — write it after the more structured methods.

MVP set:

- 1 × method overview (`schema_overview`) — early maladaptive schemas, modes, the
  relational/attachment lens.
- 2–3 × psychoeducation pieces (the inner critic; where schemas come from;
  attachment styles in plain language).
- 2–3 × intervention protocols (schema identification; chair-work — set
  `requires_supervision: true`; compassionate reframe of the inner critic).
- 2 × homework (self-compassion practice; schema-trigger diary).

Primary sources to anchor: Young JE, Klosko JS, Weishaar ME, *Schema Therapy*, 2003;
attachment literature (e.g. Wallin DJ, *Attachment in Psychotherapy*, 2007).
