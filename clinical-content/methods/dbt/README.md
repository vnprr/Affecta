# DBT content (write second)

Author DBT documents here (one `<id>.md` per document, from `../../templates/`).
Required before earlier (CBT) set is complete — see `../../METHODS_ROADMAP.md` §2.

MVP set:

- 1 × method overview (`dbt_overview`) — dialectics, four skills modules,
  validation-first stance for `borderline_traits_possible`.
- 3 × psychoeducation pieces (window of tolerance; primary vs secondary emotions;
  acceptance *and* change).
- 4 × intervention protocols, one per module (mindfulness; distress tolerance;
  emotion regulation; interpersonal effectiveness / DEAR MAN).
- 3 × homework (diary card; opposite-action; paced breathing / TIP).

Primary source to anchor: Linehan MM, *DBT Skills Training Manual*, 2nd ed., 2014.
